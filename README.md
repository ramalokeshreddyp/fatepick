**FatePick** is a high-performance, precision-driven web application designed to eliminate selection bias and organizational overhead in classrooms, workshops, and professional environments. Built with a focus on speed, privacy, and mathematical fairness, it provides a streamlined suite of tools for single selection, team formation, and complex topic allocation.

## üõ† Setup & Local Execution

FatePick uses TypeScript (TSX). To run it locally, you must use a development server that supports TypeScript transformation.

### 1. Install Dependencies
Open your terminal in the project directory and run:
```bash
npm install
```
*Note: If you encounter peer dependency errors, run `npm install --legacy-peer-deps`.*

### 2. Start Development Server
Run the following command to start the app:
```bash
npm run dev
```
The application will automatically open at [http://localhost:3000](http://localhost:3000).
